# java

A minimal Java project that demonstrates how to compile and run a simple `main` class using `javac` and `java`.

Build & run
 - Make the build script executable and run it:

```bash
chmod +x build.sh
./build.sh
```

Requirements
 - A JDK with `javac` and `java` on your PATH.

Files
 - [src/main/java/com/example/App.java](src/main/java/com/example/App.java#L1-L7): sample application
 - [build.sh](build.sh#L1-L10): compile & run script
